Semiotic Visual Regression Test Suites

This is the test suite for visual regression testing of the Semiotic data visualization library. Each suite tests a different frame type with various configurations and rendering modes.

Test Suites

  • XYFrame Test Suite
    Tests line charts, area charts, scatter plots, and combo charts in both SVG and Canvas modes
  • OrdinalFrame Test Suite
    Tests bar charts (vertical/horizontal/stacked), pie charts, timelines, and swarm plots
  • NetworkFrame Test Suite
    Tests force-directed networks, tree layouts, treemaps, sankey diagrams, chord diagrams, and more
  • Legacy Examples
    Original integration test examples

Running the Tests

To run the Playwright visual regression tests:

npm run test:dist

To generate new baseline snapshots:

npm run test:dist -- --update-snapshots
