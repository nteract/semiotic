`<StreamXYFrame />` is a wrapper around StreamXYFrame that makes its height and/or width responsive to changes in the size of its container. It has all the attributes of StreamXYFrame along with the following:

## Responsive Attributes

### responsiveWidth { _bool_ }

Whether the width (`size[0]`) changes on container size change to match the size of the container.

### responsiveHeight { _bool_ }

Whether the height (`size[1]`) changes on container size change to match the size of the container.

**Note:** `ResponsiveMinimapXYFrame` has been removed. Use `MinimapChart` with the `width` and `height` props instead. You can use a `ResizeObserver` or CSS container queries to achieve responsive behavior.
