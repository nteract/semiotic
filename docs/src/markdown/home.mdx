# Semiotic

**Welcome to Semiotic v3.**

Looking for 1.x documentation? Visit https://semioticv1.nteract.io.

## A React data visualization library for charts, networks, and beyond.

Semiotic gives you standard charts (line, bar, scatter, area) with sensible
defaults — plus the visualizations that other React charting libraries don't
offer: network graphs, streaming data, coordinated views, and statistical
summaries.

### What sets Semiotic apart

- **Network visualization** — `ForceDirectedGraph`, `SankeyDiagram`, `ChordDiagram`, `TreeDiagram`, `Treemap`, `CirclePack` as React components with the same clean API as `LineChart` - [Guide →](/guides/network-charts-hoc)
- **Streaming data** — Canvas-rendered realtime charts with a ref-based push API. Visual encodings for change: old data fades (decay), new data flashes (pulse), stale feeds are flagged - [Guide →](/charts/realtime-line-chart)
- **Coordinated views** — `LinkedCharts` for cross-highlighting and brushing-and-linking between any charts; `ScatterplotMatrix` with crossfilter brushing - [Guide →](/features/linked-charts)
- **Statistical charts** — `BoxPlot`, `ViolinPlot`, `SwarmPlot`, `Histogram`, `RidgelinePlot` for distribution analysis - [Guide →](/guides/ordinal-charts-hoc)

### All 28 chart components

- **XY Charts**: `Scatterplot`, `LineChart`, `AreaChart`, `StackedAreaChart`, `BubbleChart`, `Heatmap` - [Guide →](/guides/xy-charts-hoc)
- **Ordinal Charts**: `BarChart`, `StackedBarChart`, `GroupedBarChart`, `SwarmPlot`, `BoxPlot`, `Histogram`, `ViolinPlot`, `DotPlot`, `PieChart`, `DonutChart` - [Guide →](/guides/ordinal-charts-hoc)
- **Network Charts**: `ForceDirectedGraph`, `ChordDiagram`, `SankeyDiagram`, `TreeDiagram`, `Treemap`, `CirclePack` - [Guide →](/guides/network-charts-hoc)
- **Realtime Charts**: `RealtimeLineChart`, `RealtimeTemporalHistogram`, `RealtimeSwarmChart`, `RealtimeWaterfallChart`, `RealtimeHeatmap`

### Two levels of control

**Chart components** (recommended) give you sensible defaults, built-in
tooltips, and automatic legends. Every chart accepts a `frameProps` prop
to access the full Frame API without leaving the simpler interface.

**Frame components** (`StreamXYFrame`, `StreamOrdinalFrame`,
`StreamNetworkFrame`) give you complete control over rendering,
interaction, and layout when you need it.

## Getting Started

### Installation

Install and save `semiotic` to your project with `yarn` or `npm`.

```bash
yarn add -E semiotic
```

or

```bash
npm i -SE semiotic
```

### Usage

Import higher-order components for quick charts:

```jsx
import { Scatterplot, BarChart, LineChart } from "semiotic"

function MyChart() {
  return (
    <Scatterplot
      data={[
        { x: 1, y: 5, category: "A" },
        { x: 2, y: 8, category: "B" }
      ]}
      xLabel="X Axis"
      yLabel="Y Axis"
      colorBy="category"
    />
  )
}
```

Or import frame components for advanced customization:

```jsx
import { StreamXYFrame, StreamOrdinalFrame, StreamNetworkFrame } from "semiotic"
```

Alternatively, you can use semiotic's bundled version available on unpkg.com.

```html
<script type="module">
  import { Scatterplot, BarChart } from "https://unpkg.com/semiotic?module"
</script>
```
### Explore
