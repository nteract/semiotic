## Semiotic is a data visualization framework for React.

Semiotic provides two ways to create charts:

### Higher-Order Chart Components (Recommended)

Simplified components with intuitive APIs for common chart types:

- **XY Charts**: `Scatterplot`, `LineChart`, `AreaChart`, `BubbleChart`, `Heatmap` - [Guide →](/guides/xy-charts-hoc)
- **Ordinal Charts**: `BarChart`, `StackedBarChart`, `SwarmPlot`, `BoxPlot`, `DotPlot` - [Guide →](/guides/ordinal-charts-hoc)
- **Network Charts**: `ForceDirectedGraph`, `ChordDiagram`, `SankeyDiagram`, `TreeDiagram` - [Guide →](/guides/network-charts-hoc)

These components offer smart defaults, built-in tooltips, and an easy-to-learn API.

### Frame Components (Advanced)

Powerful, flexible frame types for custom visualizations:

- `XYFrame` - XY data (line charts, scatterplots, area charts)
- `OrdinalFrame` - Categorical data (bar charts, violin plots, parallel coordinates)
- `NetworkFrame` - Topological and network data (flow diagrams, hierarchies)

Frames provide complete control over every aspect of your visualization.

## Getting Started

### Installation

Install and save `semiotic` to your project with `yarn` or `npm`.

```bash
yarn add -E semiotic
```

or

```bash
npm i -SE semiotic
```

### Usage

Import higher-order components for quick charts:

```jsx
import { Scatterplot, BarChart, LineChart } from "semiotic"

function MyChart() {
  return (
    <Scatterplot
      data={[
        { x: 1, y: 5, category: "A" },
        { x: 2, y: 8, category: "B" }
      ]}
      xLabel="X Axis"
      yLabel="Y Axis"
      colorBy="category"
    />
  )
}
```

Or import frame components for advanced customization:

```jsx
import { XYFrame, OrdinalFrame, NetworkFrame } from "semiotic"
```

Alternatively, you can use semiotic's bundled version available on unpkg.com.

```html
<script type="module">
  import { Scatterplot, BarChart } from "https://unpkg.com/semiotic?module"
</script>
```
