`<MinimapChart />` is a high-level HOC that renders a main line chart alongside a smaller overview chart with a d3-brush for zooming into a region. It replaces the legacy `MinimapXYFrame`.

## Basic Usage

```jsx
import { MinimapChart } from "semiotic"

<MinimapChart
  data={data}
  xAccessor="date"
  yAccessor="value"
  lineBy="series"
  colorBy="series"
  curve="monotoneX"
  width={700}
  height={300}
  margin={{ left: 50, top: 10, bottom: 40, right: 20 }}
  minimap={{
    height: 50,
    margin: { left: 50, top: 0, bottom: 10, right: 20 },
  }}
/>
```

## Minimap Configuration

### minimap { _object_ }

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `height` | number | 60 | Height of the overview chart |
| `margin` | object | Matches main margin | Margin for the overview chart |
| `lineStyle` | function | Thinner main style | Line style override for overview |
| `showAxes` | boolean | false | Show axes in overview |
| `background` | string | none | Background color for overview |
| `brushDirection` | "x" \| "y" | "x" | Brush direction |

## Controlled Brush

Use `brushExtent` and `onBrush` for controlled state:

```jsx
const [extent, setExtent] = useState(null)

<MinimapChart
  data={data}
  xAccessor="step"
  yAccessor="value"
  brushExtent={extent}
  onBrush={setExtent}
  minimap={{ height: 50 }}
/>
```

## All Props

MinimapChart accepts all LineChart-like props (`data`, `xAccessor`, `yAccessor`, `lineBy`, `colorBy`, `colorScheme`, `curve`, `lineWidth`, `fillArea`, `showPoints`, `enableHover`, `tooltip`, etc.) plus:

- `minimap` — overview chart configuration (see above)
- `renderBefore` — show minimap above the main chart (default: false)
- `brushExtent` — controlled brush extent `[min, max]`
- `onBrush` — callback when brush changes
- `frameProps` — additional StreamXYFrame props for the main chart
